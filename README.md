# Simulador de Cadastro e Crédito
Projeto criado com a intenção de aprofundar os estudos e criar um menu como o de um banco que simule empréstimos. Neste projeto eu explorei estruturas de decisão e menus interativos utilizando Python 3.14.4 

### 🛠️ Funcionalidades
- Cadastro básico de usuário.
- Verificação de idade.
- Menu de alteração de dados usando `match-case`.
- Cálculo de aprovação de empréstimo (limite de 30% da renda).

### 📚 O que aprendi:
- Entrada e saída de dados (`input`/`print`).
- Manipulação de tipos (`int`, `float`, `str`).
- Controle de fluxo e lógica condicional (`if`, `else`, `match-case`).
- Importação da biblioteca Sys (`sys`).

### 🚀 Como rodar o projeto:
1. Certifique-se de ter o Python instalado.
2. Baixe o arquivo `simulador_credito.py`.
3. Instale uma IDE como PyCharm e execute o arquivo .py ou você pode executar via terminal utilizando o comando: `python simulador_credito.py`.
